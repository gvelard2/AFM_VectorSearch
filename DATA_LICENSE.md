# Data License and Provenance

## Sample dataset

The `.ibw` files included in `Data/` were collected at the University of California, Berkeley using an
Asylum AFM. They are provided as reference inputs for testing the
ingestion pipeline and are **not** intended as a curated scientific dataset.

license. You are free to share and adapt the data for any purpose, provided

**Suggested citation for the data:**

> Velarde, G., 2026. AFM Similarity Search — sample dataset. University of California, Berkeley.
> Available at: https://github.com/gvelard2/AFM_VectorSearch

## User-contributed data

Data ingested by end users (i.e. `.ibw` files uploaded via the API or UI) remain
the property of the uploading researcher. This software does not claim any rights
over user-contributed data. Administrators deploying this system are responsible
for establishing and communicating their own data governance policies.

## Instrument metadata

Instrument metadata embedded in `.ibw` files (scan parameters, operator notes,
timestamps) is reproduced as-is for provenance purposes. It does not constitute
an endorsement by instrument manufacturers.
